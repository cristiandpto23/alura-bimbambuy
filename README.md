# BimBam Buy AI Assistant

## Descripción

BimBam Buy AI Assistant es un agente inteligente basado en la técnica Retrieval-Augmented Generation (RAG), desarrollado en Python con LangChain y Gemini.

La aplicación permite responder preguntas en lenguaje natural sobre la documentación oficial de BimBam Buy, utilizando como fuente de información un conjunto de documentos PDF.

---

## Tecnologías utilizadas

- Python 3.12
- Streamlit
- LangChain
- Google Gemini
- FAISS
- Sentence Transformers
- PyPDF
- Python Dotenv
